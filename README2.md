# 🌍 Air Quality Analysis

## 📌 Overview
This project analyzes air pollution data across multiple cities using Python and data science techniques.

## 📊 Features
- City-wise pollution analysis
- Pollutant comparison
- Machine Learning (Linear Regression)

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📁 Dataset
Air Quality Dataset (12000 rows)

## 📈 Results
- PM10 is most critical pollutant
- Weak correlation observed
